# VTT jQuery Slider
A responsive and a11y friendly jQuery slider.

## Usage

1. Include jQuery:

	```html
    <script src="path/to/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="path/to/jquery.slider.js"></script>
	```

3. Call the plugin:

	```javascript
	$("#element").slider({
		// config
	});
	```
